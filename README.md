## Stack

Frontend :

- React.js
- Axios
- Stream Chat
- Universal cookies

Backend :

- Node.js
- Express
- dotenv
- Twilio
- Get stream

## Features

- Direct and Group Chats,
- Emojis & Reactions,
- Built-in Gif support,
- Ability to edit & delete messages,
- Specialized commands, and much more,
- Fully responsive
- Mobile notifications
